# **Game Statistics**

The marketing team is asking for some stats about our Fantasy Quest game servers. They'd like to know the average number of players across our 4 servers.

- **Assignment**
  Debug the code so that it correctly prints the solution to the math problem.
  The syntax is wrong. Why is it treating the numbers like plain text?
- **Tip**
  [Strings](<https://en.wikipedia.org/wiki/String_(computer_science)>) (plain text) are surrounded in quotations but [integers](https://en.wikipedia.org/wiki/Integer) (numbers) are not!
